# 机密档案 · Classified Dossier —— 视觉重设计规范

**日期**: 2026-06-14  
**状态**: 设计已确认，待实现  

---

## 1. 概述

### 1.1 目标

将 Forigen（西方马克思主义思想脉络可视化平台）的视觉设计从当前的"深空科技档案馆"风格全面改造为**"机密档案"美学**，消除 AI 生成感，营造历史文献的肌理与时代感。

### 1.2 核心隐喻

> 一份被不同机构反复批注、层层加密的**机密思想档案**——摆在档案室桌上的 19 世纪古地图地球仪，上面有不同颜色的钢笔墨水批注。

### 1.3 设计原则

- **档案物质感**：一切都像可以触摸的纸张、墨水、黄铜
- **制度美学**：打字机字体、档案编号、红章、机密标记
- **机械交互**：交互像真实的档案操作——盖章、翻页、打字机敲击
- **高对比张力**：黑红灰主导，暖纸底缓冲，戏剧性的视觉节奏

## 2. 色彩系统

### 2.1 从暗色到亮色的全局翻转

```
当前                 →    改造后
#0a0a1a (深空黑)     →    #f2ede4 (暖纸底)
#0d1a2d (暗面板)     →    #e8e0d4 (档案卡片)
#1a3a5c (暗边框)     →    #c0b8ac (档案分隔线)
#4fc3f7 (科技青)     →    #9e2a2b (机构红章)
#ffffff (白文字)     →    #1a1a1a (黑墨水)
```

### 2.2 完整色板

| 角色 | 色值 | 用途 |
|------|------|------|
| 纸底 `--paper` | `#f2ede4` | 页面背景、全局底色 |
| 面板 `--surface` | `#e8e0d4` | 卡片、面板、下拉菜单 |
| 悬停 `--surface-hover` | `#ddd5c8` | 悬停态、选中行 |
| 主文字 `--text-primary` | `#1a1a1a` | 标题、姓名、正文 |
| 辅文字 `--text-secondary` | `#5a4a3a` | 描述、日期、辅助信息 |
| 元数据 `--text-muted` | `#8b7355` | 标签、档案编号、注释 |
| 边框 `--border` | `#c0b8ac` | 面板边框、分隔线 |
| 红章 `--stamp-red` | `#9e2a2b` | 机密标记、选中态、强调操作 |
| 铜金 `--brass` | `#d4a03a` | 黄铜细节、经纬线、装饰 |
| 墨绿 `--ink-green` | `#4a6a5a` | 辅助标记、少数流派 |

### 2.3 流派颜色（SCHOOL_COLORS）

保留现有的 57 色流派颜色系统，但颜色映射调整为**墨水色感**：
- 适当降低饱和度，模拟钢笔墨水在纸上的呈色
- 在暗色纸上鲜艳的颜色，在暖纸底上需要调暗 10-15% 以保持可读性
- 每个颜色增加对应"渗透扩散"的半透明版本（opacity 0.15-0.25），用于墨迹批注的羽化边缘

## 3. 字体系统

### 3.1 字体栈

| 层级 | 英文 | 中文 | 用途 |
|------|------|------|------|
| Display | Playfair Display | 思源宋体 (Noto Serif SC) | 思想家姓名、面板标题 |
| Body | Georgia / PT Serif | 思源宋体 / 宋体 | 正文、引用、思想描述 |
| Mono | Courier New / IBM Plex Mono | 仿宋 (FangSong) | 档案编号、标签、元数据 |

### 3.2 加载方式

通过 Google Fonts 加载 Playfair Display，思源宋体通过系统字体栈或 CDN 加载。Courier New 使用系统自带。

### 3.3 排版规则

- 思想家中文名：思源宋体 Bold，14-18px
- 思想家英文名：Playfair Display Bold，16-20px
- 思想正文：Georgia / 思源宋体 Regular，11-12px，行高 1.7
- 档案编号/标签：Courier New / 仿宋，8-10px，letter-spacing: 0.05-0.1em
- 机密戳/红章文字：Courier New Bold，7-9px，全大写，letter-spacing: 0.1em
- **不再使用系统无衬线字体**（Segoe UI、Microsoft YaHei、PingFang SC 等）

## 4. 3D 地球：古地图风格

### 4.1 地球本体

- **大陆颜色**：从真实卫星纹理改为深褐色单色（`#8b6f4e`），半透明叠加在球体上
- **海洋颜色**：暖羊皮纸渐变（`#f0e8d0` → `#d4c098` → `#a89068`），从受光面到背光面
- **经纬线**：黄铜色虚线（`#8b6f4e`，opacity 0.3-0.4），使用 dashed stroke
- **大气光晕**：移除当前的 Fresnel 蓝色辉光，改为极淡的墨晕效果或直接移除
- **星空背景**：移除 2000 颗 stars，改为纯纸底 + 整体页面纸张纹理

### 4.2 地球材质

- 使用自定义 shader 或纹理贴图实现羊皮纸质感
- 可选：在球体上叠加微弱的 feTurbulence 噪点纹理（类似纸张纤维）
- 可选：黄铜色子午线环（装饰性椭圆 arc）

### 4.3 国家边界

- 当前 GeoJSON 边界线 (#2a5078) 改为深褐色（#6b5030），opacity 0.3-0.4
- 更像古地图上的手绘边界而非现代政治版图

## 5. 思想家节点：墨迹批注

### 5.1 节点形态

每个思想家在地球上表现为**钢笔墨点**：
- **内层**：实心圆，流派颜色（不透明度 0.75-0.85），直径 4-9px（随距离缩放）
- **扩散层**：同色半透明外晕（opacity 0.08-0.18），模拟墨水在纸上的渗透
- **附加扩散层**：更大的羽化圆（opacity 0.04-0.12），墨水向纸张纤维的扩散

### 5.2 标签

- 节点上方显示手写体风格的姓名标签
- 使用 Georgia italic（英文）或楷体/仿宋（中文），营造手写批注感
- 选中节点：标签加下划线（红章色 #9e2a2b），模拟档案管理员划重点

### 5.3 选中态

- 内层墨点颜色加深
- 扩散层范围扩大（+50%）
- 姓名标签加红色下划线
- 可能加一个黑色虚线外圈（类似档案中的圈注标记）

### 5.4 高亮态（Topic/School 筛选）

- 匹配节点：墨点加深、扩散明显
- 非匹配节点：墨点极淡（opacity 0.06-0.1），退入背景如同褪色墨迹

## 6. UI 组件改造

### 6.1 总则

所有面板从"半透明暗色 HUD 叠加层"改为"档案文件夹/卷宗卡片"风格：
- 背景：`#e8e0d4` 或 `#f2ede4`，不再是半透明暗色
- 边框：`#c0b8ac` 实线 1px
- 阴影：轻微纸质阴影（0 1px 4px rgba(0,0,0,0.08)），不是发光
- 可选装饰：面板角落的淡色机密印章水印

### 6.2 详情面板 (DetailPanel)

- 顶部档案条：`DOSSIER NO. {born}-{id} · CLASSIFIED`（Courier New，8px）
- 右上角：淡色圆形 `TOP SECRET` 印章水印，旋转 -12°
- 思想家姓名：Playfair Display + 思源宋体，大号衬线标题
- 生卒年/流派：Courier New 等宽标签行，用 `·` 分隔
- 引用文字：Georgia italic，左侧有红色竖线（类似引用栏）
- 标签 badges：打字机风格——黑底白字或红框黑字

### 6.3 时间轴 (Timeline / EraTimeline)

- 当前年份指示器：红色圆点 `#9e2a2b`（非青色滑块）
- 时间轴轨道：棕色细线 `#8b7355`
- 刻度标记：Courier New，7px，`#8b7355`
- Era 色条降低饱和度以适应暖纸底色

### 6.4 搜索栏 (SearchBar)

- 边框：打字机风格输入框——`#8b7355` 边框 + 闪烁光标
- Placeholder：`> SEARCH THINKER...`（Courier New，模拟命令行/打字机提示符）
- 下拉结果：档案索引卡片风格，每行带编号

### 6.5 主题/流派面板 (TopicPanel / SchoolPanel)

- 下拉面板：档案文件夹风格，选项前有对应颜色的墨点标记
- 选中指示：打字机 `[X]` 标记（替代勾选图标）

### 6.6 数据工具 (DataToolbar)

- 紧凑的打字机标签风格
- Import/Export 按钮：Courier New 字体，边框样式

## 7. 动效系统：机械档案

### 7.1 动效原则

- **果断、有力、不可撤销**——没有弹性缓动，没有淡入淡出
- **隐喻来源**：打字机敲击、印章按压、档案抽屉推动
- **时长**：0.1s–0.3s，大部分在 0.15s
- **缓动函数**：`cubic-bezier(0.3, 0, 1, 1)` 或 `steps()` 关键帧

### 7.2 关键动效

| 场景 | 动效名称 | 实现 | 时长 |
|------|----------|------|------|
| 选中思想家 | STAMP PRESS | scale(0.95)→scale(1.02)→scale(1) | 0.15s |
| 面板打开 | FILE SLIDE | translateX(100%)→translateX(0)，硬切缓动 | 0.25s |
| 面板关闭 | FILE SLIDE OUT | translateX(0)→translateX(100%) | 0.2s |
| 文字浮现 | TYPEWRITER | clip-path 从左到右揭露 + 光标闪烁 | 0.3-0.8s |
| 标签切换 | CLASSIFY MARK | 瞬时切换 + 墨水渗透扩散效果 | 0.1s |
| 筛选高亮 | INK BLEED | 墨点扩散层 scale 动画 | 0.2s |

### 7.3 CSS 实现

```css
/* 印章按压 */
@keyframes stamp-press {
  0% { transform: scale(1); }
  50% { transform: scale(0.95); }
  70% { transform: scale(1.02); }
  100% { transform: scale(1); }
}

/* 文件滑入 */
@keyframes file-slide-in {
  from { transform: translateX(100%); }
  to { transform: translateX(0); }
}

/* 打字机揭露 */
@keyframes typewriter-reveal {
  from { clip-path: inset(0 100% 0 0); }
  to { clip-path: inset(0 0 0 0); }
}
```

## 8. 纹理与氛围

### 8.1 纸张纹理

- 全局 SVG feTurbulence 噪点叠加，opacity ~4%
- 频率 0.65-0.7，4 个八度，模拟纸张纤维
- 应用于 `body::before` 伪元素或独立叠加层

### 8.2 暗角 (Vignette)

- 全局 `box-shadow: inset 0 0 60px rgba(139,115,85,0.12)` 内阴影
- 模拟档案边缘泛黄、中心略微明亮
- 边缘比中心暗 5-8%

### 8.3 印章水印

- 大尺寸（50-60px）圆形机密印章
- 低透明度（opacity 0.1-0.2）
- 置于面板角落，旋转 -12° 至 -18°
- 文字：`TOP SECRET`、`ARCHIV`、`GEHEIM`

## 9. 连线与关系线

### 9.1 思想家之间的连线 (ConnectionLines)

- 当前：发光的流动弧线（科技感）
- 改造：细虚线或点划线（`#8b7355`），类似档案中的手绘关系连线或地图测量线
- 流动动画：节点沿虚线移动，模拟墨水流向（替代发光流动）
- 选中思想家时：关联连线颜色加深或变为红章色虚线

## 10. 实现策略

### 10.1 CSS 变量化

在 `src/index.css` 的 `:root` 块中定义所有设计 token：

```css
:root {
  --paper: #f2ede4;
  --surface: #e8e0d4;
  --surface-hover: #ddd5c8;
  --text-primary: #1a1a1a;
  --text-secondary: #5a4a3a;
  --text-muted: #8b7355;
  --border: #c0b8ac;
  --stamp-red: #9e2a2b;
  --brass: #d4a03a;
  --ink-green: #4a6a5a;
  --font-display: 'Playfair Display', 'Noto Serif SC', serif;
  --font-body: 'Georgia', 'Noto Serif SC', serif;
  --font-mono: 'Courier New', 'FangSong', monospace;
}
```

### 10.2 改造顺序（建议）

1. **CSS 变量 + 全局样式**：定义变量，改 body 背景和字体
2. **3D 地球**：重写着色器/纹理，移除大气光晕和星空
3. **思想家节点**：替换光晕圆点为墨迹批注风格
4. **UI 面板**：逐个改造为档案文件夹风格
5. **时间轴 + 搜索栏**：统一档案风格
6. **动效**：替换所有 ease-out 为机械动效
7. **纹理 + 水印**：添加纸张质感和印章水印

### 10.3 兼容性

- 流派颜色系统（57 色）保留，仅调整饱和度和增加半透明版本
- 3D 交互保留（OrbitControls、节点点击、缩放自适应）
- 中英双语保留，字体选择针对双语优化
- 数据加载逻辑不变（仅视觉层改造）
