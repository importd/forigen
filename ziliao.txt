## 邓晓芒、赵林《西方哲学史》核心人物档案

### 一、古希腊罗马哲学（赵林撰写）

#### 1. 苏格拉底（Socrates）

| 模块 | 内容 |
|------|------|
| **基础档案** | 姓名：苏格拉底（Σωκράτης）；生卒：约前470年–前399年；国籍：古希腊雅典；学术身份：哲学家、伦理学家；学派归属：无正式学派（苏格拉底学派创始人） |
| **核心思想** | ① **"认识你自己"**（Γνῶθι σεαυτόν）：德尔斐神庙箴言的哲学化，强调自我反思是智慧起点；出自柏拉图《申辩篇》等对话录；从宗教诫命演变为理性主义认识论基础；误区：并非主张相对主义，而是追求普遍定义。<br>② **"美德即知识"**（ἀρετή = ἐπιστήμη）：道德行为以真知识为前提，无知即恶；出自柏拉图《美诺篇》《普罗泰戈拉篇》；被亚里士多德修正为"实践智慧"；误区：非智力主义，而是强调理性对欲望的统御。<br>③ **苏格拉底方法/助产术**（μαῖευτική）：通过诘问揭露对方矛盾，引导自主发现真理；出自柏拉图《泰阿泰德篇》；影响柏拉图辩证法与黑格尔否定辩证法；误区：非诡辩，而是追求定义而非胜利。<br>④ **灵魂不朽与关怀**：灵魂高于肉体，哲学是"练习死亡"；出自《斐多篇》；影响新柏拉图主义与基督教灵魂观；误区：非禁欲主义，而是价值排序。 |
| **代表著作** | 苏格拉底本人无著作，思想通过**柏拉图对话录**传世：<br>①《申辩篇》（Apology，约前399年）：法庭辩护词，"未经审视的生活不值得过"——命题：哲学与城邦的冲突；背景：雅典民主审判。<br>②《克里托篇》（Crito）：公民服从法律义务——命题：契约论雏形；背景：苏格拉底拒逃狱。<br>③《斐多篇》（Phaedo）：灵魂不朽论证——命题：二元论与回忆说；背景：临终场景。<br>④《会饮篇》（Symposium）：爱欲阶梯——命题：美的本体论；背景：雅典社交文化。 |
| **影响网络** | **师承**：阿那克萨戈拉（Nous概念）、自然哲学家（追问本原传统）；**批判**：智者派（普罗泰戈拉"人是万物的尺度"）；**继承**：柏拉图（理念论系统化）、亚里士多德（伦理学深化）、斯多葛派（理性主义伦理）、基督教（灵魂不朽）；**文献证据**：第欧根尼·拉尔修《名哲言行录》卷二；柏拉图对话录为 primary source。 |

#### 2. 柏拉图（Plato）

| 模块 | 内容 |
|------|------|
| **基础档案** | 姓名：柏拉图（Πλάτων）；生卒：约前428/427年–前348/347年；国籍：古希腊雅典；学术身份：哲学家、学园（Academy）创办者；学派归属：柏拉图主义/理念论 |
| **核心思想** | ① **理念论（Theory of Forms）**：可感世界分有/模仿超验的理念世界；出自《理想国》《斐多篇》；新柏拉图主义（普罗提诺"太一"流溢）与基督教神学（奥古斯丁）的继承；误区：理念非"概念"或"心理表象"，而是独立存在的本体。<br>② **回忆说（Anamnesis）**：学习是灵魂对前世所见的理念的回忆；出自《美诺篇》《斐多篇》；影响奥古斯丁光照说与笛卡尔天赋观念；误区：非经验论，而是先验论。<br>③ **洞穴寓言**：感官世界是洞穴中的影子，哲学是转向光明的上升；出自《理想国》第七卷；影响西方认识论等级结构（如黑格尔主奴辩证法）；误区：非政治隐喻，而是认识论-本体论双重结构。<br>④ **灵魂三分说**：理性（λογιστικόν）、激情（θυμοειδές）、欲望（ἐπιθυμητικόν）；出自《理想国》第四卷；影响弗洛伊德本我-自我-超我；误区：非生理学，而是功能心理学。<br>⑤ **哲学家王**：政治权力与哲学智慧的结合；出自《理想国》；影响波普尔批判（乌托邦工程）与罗尔斯正义论；误区：非极权主义，而是知识统治。 |
| **代表著作** | ①《理想国》（Republic，约前380–360年）：正义、理念论、哲学家王——命题：正义是灵魂各部分的和谐；背景：雅典民主衰落与僭主政治。<br>②《斐多篇》（Phaedo）：灵魂不朽、理念论——命题：形式因与目的因；背景：苏格拉底之死后的哲学辩护。<br>③《巴门尼德篇》（Parmenides）：理念论的自我批判——命题：一与多的辩证法；背景：对理念论内在矛盾的揭示（影响黑格尔）。<<br>④《蒂迈欧篇》（Timaeus）：宇宙生成论——命题：造物主以理念为模型创造世界；背景：自然哲学与数学化宇宙。 |
| **影响网络** | **师承**：苏格拉底（对话法、伦理追问）；**批判**：巴门尼德（存在论）、赫拉克利特（流变）；**继承**：亚里士多德（批判性继承，形式因）、新柏拉图主义（普罗提诺）、奥古斯丁（基督教柏拉图主义）、文艺复兴费奇诺、康德（理念 regulate 知识）；**文献证据**：亚里士多德《形而上学》A卷对柏拉图理念论的批判；第欧根尼·拉尔修《名哲言行录》卷三。 |

#### 3. 亚里士多德（Aristotle）

| 模块 | 内容 |
|------|------|
| **基础档案** | 姓名：亚里士多德（Ἀριστοτέλης）；生卒：前384年–前322年；国籍：古希腊斯塔吉拉/雅典；学术身份：哲学家、科学家、逍遥学派（Lyceum）创办者；学派归属：逍遥学派/亚里士多德主义 |
| **核心思想** | ① **实体（Οὐσία）学说**：个别事物是第一实体，种属是第二实体；出自《范畴篇》《形而上学》Z卷；经院哲学（托马斯·阿奎那）与近代实体论（笛卡尔、斯宾诺莎）的演变；误区：非唯物主义，"形式"是现实性原则。<br>② **四因说**：质料因、形式因、动力因、目的因；出自《物理学》第二卷；影响培根"真正的归纳法"与科学方法论；误区：目的因非"意识目的"，而是自然趋向。<br>③ **形而上学（第一哲学）**：研究"作为存在的存在"（τὸ ὂν ᾗ ὄν）；出自《形而上学》Γ卷；中世纪"形而上学"（metaphysica）命名来源；误区：非"玄学"，而是本体论。<br>④ **灵魂论（De Anima）**：灵魂是肉体的形式（ἐντελέχεια），分营养、感觉、理性三层；出自《论灵魂》；影响笛卡尔身心二元论的对照；误区：非二元论，而是 hylomorphism（形质论）。<<br>⑤ **中庸之道（Mesotēs）**：德性是两极端之间的中道；出自《尼各马可伦理学》第二卷；影响阿奎那伦理学与近代美德伦理学复兴；误区：非折中主义，而是理性选择的实践智慧。 |
| **代表著作** | ①《形而上学》（Metaphysics）：实体、四因、神作为不动的推动者——命题：Being qua Being；背景：对柏拉图理念论的批判与超越。<br>②《尼各马可伦理学》（Nicomachean Ethics）：幸福、德性、实践智慧——命题：eudaimonia 是灵魂合乎德性的现实活动；背景：对苏格拉底"美德即知识"的修正。<br>③《政治学》（Politics）：人是政治的动物、政体分类——命题：城邦先于个人；背景：对柏拉图《理想国》的批判与修正。<br>④《工具论》（Organon）：逻辑学体系（范畴、命题、三段论）——命题：思维形式的规范；背景：对柏拉图辩证法的系统化。 |
| **影响网络** | **师承**：柏拉图（理念论，但批判其分离学说）；**批判**：柏拉图（理念与事物分离）、原子论者（机械论）；**继承**：逍遥学派、新柏拉图主义（通过亚历山大里亚注释）、阿拉伯哲学（阿维森纳、阿威罗伊）、托马斯·阿奎那（经院哲学综合）、近代科学革命（培根、伽利略对亚里士多德物理学的批判）；**文献证据**：亚里士多德著作自身为 primary source；阿拉伯传记传统（如 Ibn al-Nadim《群书类述》）。 |

### 二、中世纪哲学（赵林撰写）

#### 4. 奥古斯丁（Augustine）

| 模块 | 内容 |
|------|------|
| **基础档案** | 姓名：奥勒留·奥古斯丁（Aurelius Augustinus）；生卒：354年–430年；国籍：罗马帝国北非希波；学术身份：神学家、哲学家、希波主教；学派归属：基督教柏拉图主义/教父哲学 |
| **核心思想** | ① **"我错故我在"（Si fallor, sum）**：怀疑本身证明存在，比笛卡尔"我思"更早；出自《论三位一体》与《上帝之城》；影响笛卡尔普遍怀疑方法；误区：非认识论起点，而是神学论证工具。<br>② **光照说（Illumination）**：真理是上帝之光在人心中的照耀；出自《论教师》《论三位一体》；影响经院哲学（阿奎那对"主动理智"的修正）；误区：非神秘主义，而是认识论的神学框架。<br>③ **时间的主观性**：时间是心灵的延伸（distentio animi）；出自《忏悔录》第十一卷；影响柏格森时间哲学与海德格尔存在与时间；误区：非唯心论，而是对永恒上帝视角的时间批判。<br>④ **原罪与恩典**：人类自由意志因亚当堕落而受损，救赎依赖上帝恩典；出自《论意志的自由选择》《上帝之城》；影响宗教改革（路德、加尔文）；误区：非宿命论，而是恩典与自由的张力。<br>⑤ **两城论**：上帝之城与地上之城的历史对立；出自《上帝之城》；影响中世纪政治神学与现代世俗化理论；误区：非二元政治纲领，而是末世论框架。 |
| **代表著作** | ①《忏悔录》（Confessions，约397–400年）：自传体神学——命题：自我是上帝的记忆与追寻；背景：摩尼教→新柏拉图主义→基督教的皈依历程。<br>②《上帝之城》（De Civitate Dei，413–426年）：历史神学——命题：罗马陷落后的基督教辩护；背景：410年罗马陷落的政治危机。<br>③《论三位一体》（De Trinitate）：神学哲学——命题：三位一体的心灵类比（记忆、理智、意志）；背景：反阿里乌派与反新柏拉图主义异端。<br>④《论教师》（De Magistro）：语言与知识——命题：基督是内在教师；背景：对符号与指称的哲学分析。 |
| **影响网络** | **师承**：新柏拉图主义（普罗提诺、波菲利）、西塞罗（修辞学与伦理学）、保罗书信；**批判**：摩尼教（善恶二元论）、佩拉纠主义（人性无罪论）；**继承**：波爱修斯（《哲学的慰藉》）、安瑟尔谟（本体论证明）、阿奎那（存在论与光照说的修正）、笛卡尔（我思传统）、帕斯卡尔（恩典神学）；**文献证据**：奥古斯丁著作集（CCSL/PL 版本）；彼得·布朗《奥古斯丁传》传记研究。 |

#### 5. 托马斯·阿奎那（Thomas Aquinas）

| 模块 | 内容 |
|------|------|
| **基础档案** | 姓名：托马斯·阿奎那（Thomas Aquinas）；生卒：约1225年–1274年；国籍：意大利/神圣罗马帝国；学术身份：神学家、哲学家、多明我会修士；学派归属：经院哲学/托马斯主义 |
| **核心思想** | ① **存在与本质的区分（Esse/Essentia）**：存在（actus essendi）是附加于本质的现实性，上帝是"自有存在"（Esse per se subsistens）；出自《论存在与本质》《神学大全》；影响海德格尔存在论差异（Sein/Seiende）；误区：非存在主义，而是存在论神学。<br>② **五路证明（Quinque Viae）**：从运动、因果、偶然、等级、目的论证明上帝存在；出自《神学大全》I, q.2, a.3；影响自然神学传统与康德批判；误区：非科学证明，而是形而上学论证。<br>③ **类比学说（Analogy）**：上帝之名以类比方式言说（既非 univocal 亦非 equivocal）；出自《神学大全》I, q.13；影响宗教语言哲学；误区：非不可知论，而是有限理性的言说方式。<br>④ **理智与意志的和谐**：理智认识善，意志趋向善；出自《神学大全》I-II；影响伦理学中理性主义与意志主义的争论；误区：非理智主义独断，而是理智优先但意志自由。<br>⑤ **自然法（Lex Naturalis）**：道德律根植于人性自然倾向，可被理性发现；出自《神学大全》I-II, q.90–94；影响格劳秀斯、洛克、现代自然权利理论；误区：非实证法，而是道德形而上学。 |
| **代表著作** | ①《神学大全》（Summa Theologica，1265–1274）：系统神学——命题：信仰与理性的综合；背景：亚里士多德哲学再发现后的基督教思想整合。<br>②《论存在与本质》（De Ente et Essentia，1252–1256）：形而上学导论——命题：存在作为最高概念；背景：对阿拉伯哲学（阿维森纳）的批判继承。<br>③《反异教大全》（Summa Contra Gentiles，1259–1265）：护教哲学——命题：理性可证明的真理与信仰真理的区分；背景：对穆斯林与犹太思想家的对话。<br>④《亚里士多德〈形而上学〉注释》：哲学注释——命题：基督教化的亚里士多德主义；背景：拉丁亚里士多德学派的兴起。 |
| **影响网络** | **师承**：亚里士多德（通过阿拉伯注释传统）、奥古斯丁（神学框架）、大阿尔伯特（亚里士多德研究）；**批判**：阿威罗伊主义（单一理智论）、奥古斯丁主义（光照说过度）；**继承**：新托马斯主义（19世纪利奥十三世《永恒之父》通谕）、马里坦（现代托马斯主义）、麦金泰尔（德性伦理学复兴）；**文献证据**：阿奎那著作集（Leonine Edition）；吉尔松《中世纪哲学精神》。 |

### 三、近代哲学（邓晓芒撰写）

#### 6. 笛卡尔（René Descartes）

| 模块 | 内容 |
|------|------|
| **基础档案** | 姓名：勒内·笛卡尔（René Descartes）；生卒：1596年–1650年；国籍：法国；学术身份：哲学家、数学家、科学家；学派归属：大陆理性主义/笛卡尔主义 |
| **核心思想** | ① **"我思故我在"（Cogito, ergo sum）**：怀疑一切后，思维活动本身证明思维者存在；出自《第一哲学沉思集》沉思二；影响康德"我思必须能够伴随我的一切表象"；误区：非三段论推理（大前提：凡思维者存在），而是直观确定性。<br>② **方法怀疑（Methodical Doubt）**：系统怀疑感官、梦境、恶魔欺骗，以寻找不可动摇的基础；出自《第一哲学沉思集》沉思一；影响现象学悬搁（epoché）；误区：非怀疑主义，而是建设性怀疑。<br>③ **心身二元论（Mind-Body Dualism）**：思维实体（res cogitans）与广延实体（res extensa）的根本区分；出自《第一哲学沉思集》沉思六；影响心身问题至今；误区：非否认身心互动（通过松果腺假说），而是本体论区分。<br>④ **天赋观念（Innate Ideas）**：上帝、自我、数学真理是心灵固有的；出自《第一哲学沉思集》沉思三；影响莱布尼茨、康德先验论；误区：非经验否定，而是对经验主义的批判。<br>⑤ **上帝存在的证明**：从"我"的不完满性推出完满者（上帝）的存在，以及从上帝完满性推出外部世界存在；出自《第一哲学沉思集》沉思三、五；影响安瑟尔谟本体论证明的笛卡尔版本；误区：非循环论证（"笛卡尔循环"争议）。 |
| **代表著作** | ①《第一哲学沉思集》（Meditationes de Prima Philosophia，1641）：形而上学奠基——命题：我思作为第一原理；背景：与经院哲学决裂，寻求新基础。<br>②《谈谈方法》（Discours de la méthode，1637）：方法论宣言——命题：四条规则（怀疑、分析、综合、枚举）；背景：科学革命时期的普遍方法追求。<br>③《哲学原理》（Principia Philosophiae，1644）：物理学与形而上学体系——命题：机械论自然观；背景：替代亚里士多德-经院自然哲学。<br>④《论灵魂的激情》（Les Passions de l'âme，1649）：伦理学与心理学——命题：激情是灵魂与身体的互动；背景：与伊丽莎白公主的通信深化。 |

#### 7. 斯宾诺莎（Baruch Spinoza）

#### 8. 莱布尼茨（Gottfried Wilhelm Leibniz）

| 模块 | 内容 |

| **基础档案** | 姓名：戈特弗里德·威廉·莱布尼茨（Gottfried Wilhelm Leibniz）；生卒：1646年–1716年；国籍：神圣罗马帝国（莱比锡）；学术身份：哲学家、数学家、外交官、博学家；学派归属：大陆理性主义/莱布尼茨主义 |

| **核心思想** | ① **自然状态与社会契约**：自然状态下人是自由平等的，私有制是不平等的根源；出自《论人类不平等的起源和基础》；影响马克思异化理论与无政府主义；误区：非历史描述，而是思想实验与批判工具。<br>② **公意（Volonté Générale）**：社会契约形成的集体意志，不同于众意（volonté de tous）；出自《社会契约论》第二卷；影响法国大革命（罗伯斯庇尔）与民主理论；误区：非极权主义，而是民主合法性基础（但存在"强迫自由"争议）。<<br>③ **自然教育（Éducation Naturelle）**：教育应顺应儿童自然发展，而非社会偏见；出自《爱弥儿》；影响现代教育理论与进步主义教育；误区：非放任教育，而是有引导的自然发展。<br>④ **文明的异化**：科学与艺术进步反而败坏道德（《论科学与艺术》）；出自《论科学与艺术》；影响法兰克福学派文化批判与海德格尔技术批判；误区：非反智主义，而是对文明病的诊断。<br>⑤ **良心与情感**：良心是内在的声音，先于理性；出自《爱弥儿》第四卷"萨瓦牧师的信仰自白"；影响康德道德情感论与存在主义；误区：非理性主义，而是对理性独断的补充。 |
| **代表著作** | ①《论人类不平等的起源和基础》（Discours sur l'origine et les fondements de l'inégalité parmi les hommes，1755）：政治哲学批判——命题：私有制是不平等的根源；背景：第戎科学院征文。<br>②《社会契约论》（Du Contrat Social，1762）：政治哲学建构——命题：公意作为主权基础；背景：对霍布斯、洛克契约论的修正。<br>③《爱弥儿》（Émile, ou De l'éducation，1762）：教育哲学——命题：自然教育培养自然人；背景：对经院教育与贵族教育的全面批判。<br>④《论科学与艺术》（Discours sur les sciences et les arts，1750）：文明批判——命题：科学艺术败坏道德；背景：第戎科学院征文获奖。 |

### 四、德国古典哲学（邓晓芒撰写）

#### 12. 康德（Immanuel Kant）

| 模块 | 内容 |
|------|------|
| **基础档案** | 姓名：伊曼努尔·康德（Immanuel Kant）；生卒：1724年–1804年；国籍：普鲁士柯尼斯堡；学术身份：哲学家、柯尼斯堡大学教授；学派归属：德国古典哲学/批判哲学/先验唯心论 |
| **核心思想** | ① **哥白尼式革命（Copernican Revolution）**：不是知识符合对象，而是对象符合知识（先验唯心论）；出自《纯粹理性批判》第二版序言；影响整个德国唯心论与分析哲学；误区：非主观唯心论（贝克莱），而是先验论（形式由主体提供，质料由物自体提供）。<<br>② **先验演绎（Transcendental Deduction）**：证明范畴对经验对象的客观有效性；出自《纯粹理性批判》先验演绎章；影响黑格尔逻辑学与胡塞尔现象学；误区：非心理学论证，而是先验论证。<br>③ **物自体（Ding an sich/Noumenon）**：不可知的本体，是现象的基础但非认识对象；出自《纯粹理性批判》先验辩证论；影响费希特（取消物自体）、叔本华（意志作为物自体）；误区：非二元论，而是认识论界限。<br>④ **定言命令（Categorical Imperative）**："只依据那些你能同时愿意它成为普遍法则的准则行动"；出自《道德形而上学基础》第二章；影响义务论伦理学与罗尔斯正义论；误区：非规则功利主义，而是形式道德法则。<br>⑤ **判断力与目的论**：自然合目的性作为反思判断力的调节性原则；出自《判断力批判》；影响黑格尔辩证法与生物学哲学；误区：非目的论证明，而是审美与目的论的判断力批判。 |
| **代表著作** | ①《纯粹理性批判》（Kritik der reinen Vernunft，1781/1787）：认识论与形而上学——命题：先天综合判断如何可能；背景：回应休谟怀疑论与莱布尼茨-沃尔夫独断论。<br>②《实践理性批判》（Kritik der praktischen Vernunft，1788）：道德哲学——命题：自由意志与道德法则的实在性；背景：对《道德形而上学基础》的体系化与对道德怀疑论的回应。<br>③《判断力批判》（Kritik der Urteilskraft，1790）：美学与目的论——命题：自然合目的性与审美判断；背景：沟通理论理性与实践理性的桥梁。<br>④《道德形而上学基础》（Grundlegung zur Metaphysik der Sitten，1785）：伦理学导论——命题：定言命令的三种公式；背景：为道德哲学寻找先验基础。 |
| **影响网络** | **师承**：莱布尼茨-沃尔夫体系（理性主义）、休谟（怀疑论的唤醒）、牛顿（自然科学）、卢梭（对人的尊重）；**批判**：莱布尼茨-沃尔夫（独断论）、休谟（怀疑论）、贝克莱（主观唯心论）；**继承**：费希特（知识学，取消物自体）、谢林（同一哲学）、黑格尔（绝对唯心论）、叔本华（意志哲学）、新康德主义（马堡学派、西南学派）、胡塞尔（先验现象学）、罗尔斯（建构主义）；**文献证据**：康德《著作集》（普鲁士科学院版）；艾利森《康德的先验唯心论》。 |

#### 13. 黑格尔（Georg Wilhelm Friedrich Hegel）

| 模块 | 内容 |
|------|------|
| **基础档案** | 姓名：格奥尔格·威廉·弗里德里希·黑格尔（Georg Wilhelm Friedrich Hegel）；生卒：1770年–1831年；国籍：德意志符腾堡/普鲁士；学术身份：哲学家、柏林大学校长；学派归属：德国古典哲学/绝对唯心论/黑格尔主义 |
| **核心思想** | ① **辩证法（Dialectic）**：正题-反题-合题（Triad）的矛盾运动，是事物发展的内在逻辑；出自《精神现象学》《逻辑学》；影响马克思辩证法、存在主义、法兰克福学派；误区：非机械三段论，而是概念自我运动的逻辑。<br>② **绝对精神（Absolute Spirit）**：世界历史的终极主体，通过自我异化与回归实现自我认识；出自《精神现象学》《哲学全书》；影响马克思历史唯物主义（头足倒置的批判）、科耶夫历史终结论；误区：非神学，而是哲学化的历史目的论。<br>③ **实体即主体（Substance as Subject）**：斯宾诺莎的实体通过费希特的自我获得主体性；出自《精神现象学》序言；影响马克思《1844年手稿》与卢卡奇《历史与阶级意识》；误区：非人格神，而是自我运动的概念。<br>④ **主奴辩证法（Master-Slave Dialectic）**：自我意识通过承认斗争确立，奴隶通过劳动获得真正的自我意识；出自《精神现象学》第四章；影响马克思劳动异化、科耶夫历史哲学、萨特《存在与虚无》；误区：非阶级斗争理论，而是意识发展史。<br>⑤ **历史哲学**：历史是理性（绝对精神）的展开，"凡是合理的都是现实的，凡是现实的都是合理的"；出自《法哲学原理》序言、《历史哲学讲演录》；影响历史主义与保守主义；误区：非现状辩护，而是理性与现实的辩证统一（合理=合乎理性）。 |
| **代表著作** | ①《精神现象学》（Phänomenologie des Geistes，1807）：意识发展史——命题：从感性确定性到绝对知识的辩证历程；背景：耶拿时期，拿破仑入耶拿当天完稿。<br>②《逻辑学》（Wissenschaft der Logik，1812–1816）：辩证逻辑体系——命题：存在论-本质论-概念论的辩证展开；背景：对康德先验逻辑的批判性超越。<br>③《法哲学原理》（Grundlinien der Philosophie des Rechts，1821）：政治哲学——命题：伦理生活（家庭-市民社会-国家）的辩证结构；背景：普鲁士复辟时期的政治哲学。<br>④《哲学全书》（Enzyklopädie der philosophischen Wissenschaften，1817）：体系总纲——命题：逻辑学-自然哲学-精神哲学的体系；背景：柏林大学讲座的系统化。 |
| **影响网络** | **师承**：康德（批判哲学）、费希特（自我哲学）、谢林（同一哲学）、斯宾诺莎（实体概念）、古希腊哲学（特别是亚里士多德）；**批判**：康德（物自体与现象的二元论）、费希特（主观唯心论）、谢林（同一哲学的非理性主义）、浪漫主义（直观优于概念）；**继承**：青年黑格尔派（费尔巴哈、马克思、鲍威尔、施蒂纳）、存在主义（克尔凯郭尔对黑格尔的批判性继承）、新黑格尔主义（克罗齐、布拉德雷）、法兰克福学派（否定辩证法）、科耶夫（历史终结论）、哈贝马斯（交往行为理论对黑格尔的修正）；**文献证据**：黑格尔《著作集》（Suhrkamp版或历史考订版）；泰勒《黑格尔》；科耶夫《黑格尔导读》。 |

#### 14. 叔本华（Arthur Schopenhauer）

#### 15. 尼采（Friedrich Nietzsche）

| 模块 | 内容 |
|------|------|
| **基础档案** | 姓名：弗里德里希·尼采（Friedrich Nietzsche）；生卒：1844年–1900年；国籍：普鲁士/德意志帝国；学术身份：古典语文学家、巴塞尔大学教授（辞职后漫游著述）；学派归属：非理性主义/生命哲学/尼采主义 |
| **核心思想** | ① **权力意志（Wille zur Macht）**：生命的本质是追求力量的增强与扩张，非叔本华的生存意志；出自《权力意志》（遗稿，但核心思想见于《查拉图斯特拉如是说》《善恶的彼岸》）；影响弗洛伊德（本能理论）、德勒兹（欲望机器）、福柯（权力知识）；误区：非政治权力，而是形而上学本体（力量增强）。<<br>② **超人（Übermensch）**：超越现存人类形态的理想，肯定生命、创造新价值；出自《查拉图斯特拉如是说》序言；影响存在主义（海德格尔《尼采》）、纳粹误用（与尼采本意无关）；误区：非种族优越论，而是自我超越的哲学理想。<br>③ **重估一切价值（Umwertung aller Werte）**：批判基督教-柏拉图主义传统的道德，建立以生命肯定为基础的新价值；出自《反基督》《权力意志》；影响后现代主义（德里达、福柯）；误区：非虚无主义，而是积极的虚无主义（虚无主义作为过渡）。<<br>④ **永恒轮回（Ewige Wiederkunft）**：假设一切事件无限重复，作为生命肯定的终极考验；出自《快乐的科学》§341、《查拉图斯特拉如是说》；影响存在主义（加缪《西西弗神话》）、德勒兹（差异与重复）；误区：非宇宙论假说，而是存在论选择。<br>⑤ **酒神精神（Dionysian）**：肯定生命的痛苦与矛盾，与日神精神（Apollonian）的秩序形成张力；出自《悲剧的诞生》；影响法兰克福学派（阿多诺否定辩证法）、后现代美学；误区：非放纵，而是对悲剧智慧的肯定。 |
| **代表著作** | ①《悲剧的诞生》（Die Geburt der Tragödie，1872）：美学与文化批判——命题：酒神与日神精神的辩证；背景：瓦格纳影响下的古典学革命。<br>②《查拉图斯特拉如是说》（Also sprach Zarathustra，1883–1885）：哲学诗——命题：超人、永恒轮回、上帝已死；背景：与瓦格纳决裂后的独立哲学表达。<br>③《善恶的彼岸》（Jenseits von Gut und Böse，1886）：道德哲学批判——命题：主人道德与奴隶道德；背景：对基督教-康德道德传统的系统批判。<br>④《权力意志》（Der Wille zur Macht，1901遗稿出版，非尼采本人编辑）：哲学体系尝试——命题：权力意志作为存在原则；背景：妹妹伊丽莎白·福斯特-尼采的编辑（存在争议）。 |
| **影响网络** | **师承**：叔本华（意志哲学、悲观主义）、瓦格纳（艺术理想）、古希腊悲剧（埃斯库罗斯、索福克勒斯）、赫拉克利特（流变哲学）；**批判**：苏格拉底（理性主义的开端）、柏拉图（理念论与基督教的前身）、基督教（奴隶道德）、康德（道德形式主义）、叔本华（否定意志）；**继承**：海德格尔（《尼采》讲座）、萨特（存在主义）、加缪（荒诞哲学）、德勒兹（《尼采与哲学》）、福柯（权力分析）、德里达（解构）、哈贝马斯（对尼采的批判性对话）、施特劳斯（政治哲学）；**文献证据**：尼采《全集》（KSA考订版）；海德格尔《尼采》；考夫曼《尼采：哲学家、心理学家、反基督者》。 |

#### 16. 马克思（Karl Marx）—— 

| 模块 | 内容 |
|------|------|
| **基础档案** | 姓名：卡尔·马克思（Karl Marx）；生卒：1818年–1883年；国籍：普鲁士/德意志帝国（后流亡英国）；学术身份：哲学家、经济学家、革命理论家；学派归属：马克思主义/历史唯物主义/政治经济学批判 |
| **核心思想** | ① **历史唯物主义（Historical Materialism）**：生产力决定生产关系，经济基础决定上层建筑；出自《德意志意识形态》（1845–1846，与恩格斯合著）、《〈政治经济学批判〉序言》（1859）；影响列宁主义、西方马克思主义、结构主义马克思主义（阿尔都塞）；误区：非经济决定论（机械论），而是"归根到底"的决定作用。<br>② **异化劳动（Entfremdung）**：资本主义下工人与劳动产品、劳动过程、类本质、他人的异化；出自《1844年经济学哲学手稿》；影响法兰克福学派（马尔库塞、弗洛姆）、存在主义马克思主义（萨特）；误区：非心理学概念，而是政治经济学批判范畴。<br>③ **剩余价值理论（Mehrwert）**：资本家剥削工人的秘密在于无偿占有剩余劳动；出自《资本论》第一卷（1867）；影响政治经济学与劳动社会学；误区：非道德控诉，而是科学分析。<br>④ **意识形态批判（Ideologiekritik）**：统治阶级的思想是占统治地位的思想；出自《德意志意识形态》；影响阿尔都塞意识形态国家机器、福柯知识-权力；误区：非虚假意识（简单否定），而是对观念物质基础的揭示。<br>⑤ **实践哲学（Praxis）**：哲学家们只是解释世界，问题在于改变世界；出自《关于费尔巴哈的提纲》第十一条；影响葛兰西实践哲学、南斯拉夫实践派；误区：非反理论，而是理论与实践的统一。 |
| **代表著作** | ①《1844年经济学哲学手稿》（Ökonomisch-philosophische Manuskripte，1844/1932出版）：早期哲学——命题：异化劳动与共产主义人本学；背景：巴黎时期对古典经济学与黑格尔哲学的批判性吸收。<br>②《德意志意识形态》（Die deutsche Ideologie，1845–1846/1932出版）：历史唯物主义奠基——命题：意识形态批判与唯物史观；背景：对青年黑格尔派（费尔巴哈、鲍威尔、施蒂纳）的清算。<br>③《共产党宣言》（Manifest der Kommunistischen Partei，1848，与恩格斯合著）：政治纲领——命题：阶级斗争与无产阶级革命；背景：1848年革命前夕的共产主义者同盟委托。<br>④《资本论》（Das Kapital，1867–1894）：政治经济学批判——命题：剩余价值与资本主义积累规律；背景：对古典经济学（斯密、李嘉图）与黑格尔逻辑学的批判继承。 |
| **影响网络** | **师承**：黑格尔（辩证法）、费尔巴哈（唯物主义人本学）、亚当·斯密（古典经济学）、李嘉图（劳动价值论）、法国社会主义（圣西门、傅立叶）、英国政治经济学；**批判**：黑格尔（唯心主义）、费尔巴哈（直观唯物主义）、蒲鲁东（小资产阶级社会主义）、拉萨尔（国家社会主义）、巴枯宁（无政府主义）；**继承**：恩格斯（共同创立马克思主义）、列宁（帝国主义论与革命实践）、卢卡奇（西方马克思主义开端）、葛兰西（文化霸权）、法兰克福学派（批判理论）、阿尔都塞（结构主义马克思主义）、萨特（存在主义马克思主义）、哈贝马斯（交往行为理论对马克思的修正）；**文献证据**：马克思恩格斯《全集》（MEGA²考订版）；麦克莱伦《马克思传》；阿尔都塞《读〈资本论〉》。 |
| **☑️ 结构主义马克思主义特有模块** | **阿尔都塞式阅读（Althusserian Reading）**：阿尔都塞在《读〈资本论〉》中提出"症候阅读"（symptomatic reading），认为马克思的《资本论》存在"认识论断裂"（1845年前后从意识形态到科学），强调其"理论反人道主义"与"多元决定论"（overdetermination）。马克思的"归根到底"决定作用被阿尔都塞修正为多元因果性。 |
| **☑️ 矛盾与多元决定论** | **多元决定论（Overdetermination）**：阿尔都塞借用弗洛伊德术语，认为社会矛盾（经济、政治、意识形态）并非单一经济决定，而是多元因素"过度决定"的复杂结构。这一概念源于对马克思"归根到底"决定论的拉康式重构，影响后马克思主义（拉克劳、墨菲）与文化研究（斯图亚特·霍尔）。 |
